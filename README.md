# Bangkit 2022 Capstone Project ( C22-PS123 ) : Rayuan Application

Improving handwriting can take a long time. Thus, we are proposing an education app to help users achieve better handwriting. The idea is to build an AI model on Android to rate handwriting via camera captures. Users need to take a photo of their handwriting with the app for an AI model to predict whether the userâ€™s handwriting is good or bad. Finally, the app will rate the handwriting with the result of the prediction given by the AI model.
